# SQL 문법

## 데이터베이스 생성

```mysql
CREATE DATABASE databasename;
```

새로운 데이터 베이스를 생성합니다.



## 데이터베이스 확인

```mysql
SHOW DATABASES;
```

현재 생성되어있는 데이터베이스를 확인합니다.



## 데이터베이스 선택

```mysql
USE databasename;
```

활성화시킬 데이터베이스를 선택합니다.
